# SimpleDrive
SimpleDrive 简单云，内存CPU占用极简/Less use in Memory and CPU

# 一款内存CPU占用极简的云盘/A cloud disk with minimal memory and CPU usage


<p>功能包括但不限于(仍在开发)：</p>





#1. 管理员登录。

#2. 用户登录

#3. 管理员添加，删除用户。添加管理员。

#4. 分享文件，文件夹.

#5. 管理员自助更新系统，管理员开启关闭注册
